## 说明

- **MCC (Mobile Country Code)**: 移动国家代码，由ITU-T分配给每个国家或地区
- **MNC (Mobile Network Code)**: 移动网络代码，标识特定运营商
- **PhonePrefix**: 手机号码前缀，用于区分不同的运营商或服务类型

## 使用方法

将此资料库中的数据复制到 `src/config/carrier_info.py` 文件中，替换原有的 `CARRIER_INFO` 字典。

## 注意事项

1. 确保MCC和MNC的值与实际情况一致
2. PhonePrefix应包含该运营商使用的所有号码前缀
3. 定期更新资料库以保持数据的准确性
